# 🌙 Midnight Memories ❤️

> *"Every memory has a story. Every story deserves to be remembered."*

A beautifully crafted interactive birthday surprise website designed to celebrate memories, laughter, friendship, and heartfelt emotions.

This project isn't just a birthday website—it's a journey through unforgettable moments, wrapped in elegant animations and a premium user experience.

---

## ✨ Features

- ❤️ Premium Animated Loader
- 🌌 Beautiful Glassmorphism UI
- ✨ Animated Background & Floating Hearts
- 🎵 Background Music Player
- 📖 Story-like Interactive Flow
- 💌 Personalized Letter
- 🕰️ Memory Timeline
- 📸 Beautiful Memory Gallery
- 😂 Inside Jokes Section
- 🎁 Birthday Surprise Gift
- ❤️ One Last Message with Typing Animation
- 🎉 Confetti Celebration
- 🔄 Restart Experience
